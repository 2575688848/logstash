## Logstash 项目运行相关

## Logstash 目录

    bin          logstash 各配置文件对应的控制脚本
        back     后端方向控制脚本
        fex      前端方向控制脚本
        inf      基础服务方向控制脚本
        sre      基础运维方向控制脚本
        
    conf         logstash 的配置文件
        back     后端方向配置文件
        fex      前端方向配置文件
        inf      基础服务方向配置文件
        sre      基础运维方向配置文件
